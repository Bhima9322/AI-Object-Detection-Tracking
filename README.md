# AI Object Detection and Tracking System 🚀

A real-time computer vision project that detects and tracks multiple objects using a webcam feed. The system intelligently categorizes detections into 'Human' and 'Object' with unique tracking IDs.

## Features ✨
* **Real-Time Detection:** Uses the YOLOv8 model for high-accuracy object detection.
* **Object Tracking:** Integrates the DeepSORT algorithm to assign and maintain unique IDs for moving objects.
* **Custom Categorization:** Dynamically labels humans and non-human objects with distinct colored bounding boxes.
* **Performance Monitoring:** Displays a live FPS (Frames Per Second) counter on the screen.

## Tech Stack 🛠️
* **Language:** Python
* **Computer Vision:** OpenCV
* **Object Detection:** YOLOv8 (Ultralytics)
* **Object Tracking:** DeepSORT Realtime
